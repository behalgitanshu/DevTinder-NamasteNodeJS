# Codemate — Frontend

This is the React + Vite client for **Codemate**, a Tinder-inspired matchmaking app. See the [root README](../README.md) for the full project overview, tech stack, and setup instructions covering both backend and frontend.

## Quick start

```bash
npm install
cp .env.example .env   # set VITE_BASE_URL to your backend URL
npm run dev
```

## Scripts

- `npm run dev` — start the Vite dev server
- `npm run build` — build for production (outputs to `dist/`)
- `npm run preview` — preview the production build locally
- `npm run lint` — run ESLint
- `npm run test` — run Vitest component tests

## Stack

React 19, Vite, Redux Toolkit, React Router, Tailwind CSS 4 + daisyUI, Axios. Full details in the [root README](../README.md#tech-stack).
